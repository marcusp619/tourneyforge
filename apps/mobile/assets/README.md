# Mobile Assets

Placeholders for Expo app assets.

Run `npx expo generate-assets` in this directory to generate default assets, or replace with your own:

- `icon.png` - 1024x1024 app icon
- `splash.png` - 1284x2778 splash screen (portrait)
- `adaptive-icon.png` - 1024x1024 adaptive icon for Android
- `favicon.png` - 48x48 favicon for web
